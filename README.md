Nina is a smart, interactive AI robot built using Python, designed to simulate human-like conversation with both text and voice interaction. This project is the first step toward building a fully functional AI-powered companion robot 🧠✨

🚀 Features
💬 Conversational AI — Chat with Nina in real-time
🔊 Voice Output — Nina speaks responses using text-to-speech
🧠 Custom Logic — Can be extended with real AI models
⚡ Lightweight & Fast — Runs locally without heavy resources
🔐 Secure Setup — API keys managed safely using environment files
🛠️ Tech Stack
🐍 Python
🗣️ pyttsx3 (Text-to-Speech)
🌐 OpenAI API 
📦 python-dotenv for secure environment management
